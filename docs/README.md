# Edge DocGraph Engine — Documentation

A local-only intelligent document-to-form engine that converts uploaded documents into editable, evidence-backed forms, learns corrected templates locally, and extracts future similar documents through fast ROI-first verification.

This folder is the complete documentation set for the project. It is organized into 18 numbered sections. Each section is self-contained and includes a `MANIFEST.md` listing its files.

---

## How to read these docs

If you are new, read in this order:

1. [`00_MASTER/MASTER_PRD.md`](00_MASTER/MASTER_PRD.md) — what must be built

## Conventions

- Each section folder contains a `MANIFEST.md` listing its files.
- Machine-readable schemas live in [`14_API_SCHEMAS`](14_API_SCHEMAS/) alongside example payloads under `EXAMPLES/`.
- The JSON schemas, TypeScript interfaces, examples, and validators must evolve together — no schema change is complete until all are updated.
