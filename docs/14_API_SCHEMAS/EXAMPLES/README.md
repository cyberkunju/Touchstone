# API Schema Examples

This directory contains synthetic, privacy-safe example JSON outputs.

Files:

- `passport_docgraph_example.json` — example DocGraph for a synthetic passport-like document.
- `invoice_form_example.json` — example editable form projection with a conflict.
- `passport_templategraph_example.json` — example TemplateGraph for a synthetic passport layout.
- `validation_report_example.json` — example validation report with a table-total conflict.
- `export_manifest_example.json` — example export package manifest.

Rules:

- These examples use fake/synthetic data.
- Do not replace these with real passport, invoice, bank, signature, MRZ, or QR payload data.
- If examples change, update the matching schema docs and regression tests.
